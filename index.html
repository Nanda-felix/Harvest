O que é ?

O Harvest ID é um aplicativo mobile que auxilia agricultores e entusiastas de jardinagem a identificarem rapidamente o estado de saúde de suas plantas. Utilizando algoritmos de inteligência artificial treinados com uma vasta base de imagens, o app é capaz de analisar fotos das plantas e indicar se elas estão saudáveis ou se apresentam sinais de doenças. Com uma interface intuitiva e simples, o Harvest ID possibilita que qualquer pessoa, mesmo sem conhecimento técnico, possa monitorar suas plantações, prevenindo problemas antes que se tornem graves.
